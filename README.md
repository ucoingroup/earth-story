## How it works

1. `daily-stories.js` generates daily positive energy stories (10 themes, 14-20 stories/day)
2. `build-story-site.js` parses stories and generates multi-language JSON data files
3. `index.html` is a single-page app that loads JSON data and renders stories
4. `deploy.js` pushes everything to GitHub Pages
